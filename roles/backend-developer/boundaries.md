# Backend Developer — Boundaries

## Allowed

- Create and modify server-side source files (controllers, services, repositories, middleware, utilities)
- Create and modify backend test files
- Modify API route definitions and configurations
- Update backend dependency configurations
- Create and modify environment variable templates (.env.example)
- Write and update API documentation

## Not Allowed

- Modify frontend/UI component files
- Create or modify database migration files (hand off to DBA)
- Modify CI/CD pipeline configurations (hand off to DevOps)
- Change infrastructure or deployment configurations (hand off to DevOps)
- Modify security policies or authentication configurations (consult Security Analyst)
- Change project-level architecture without Tech Lead approval
